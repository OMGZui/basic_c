## 科普

- If you use scanf() to read a value for one of the basic variable types we’ve discussed, precede the variable name with an &.
- If you use scanf() to read a string into a character array, don’t use an &.

```bash

-13.45e12#  0

%d  -> -13
%f  -> −13.45E12
%s  -> −13.45E12#
%c  -> -

```
